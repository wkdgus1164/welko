# welko
인하공전 모바이프로그래밍 WELKO 프로젝트


### 프로젝트 주제
외국인이 한국에 여행을 왔을 때, 국내에서 여행지나 볼거리를 모아서 보여주는 서비스

### 주요 기능
- 크롤링한 데이터를 Android 의 RecyclerView 에 보여주는 기능
- Firebase 를 통한 회원 인증 로그인 기능
- RecyclerView 의 item 을 클릭하여 해당 item 의 상세 페이지를 보여주는 기능
